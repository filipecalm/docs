## Como adicionar uma nova análise

1. Criar `analyses/[nome-do-projeto]/`
2. Gerar `product-analysis.md` seguindo o formato de saída do Prompt 4
3. Opcional: `backlog.md` com Etapa 5 em formato de issues
4. Registrar nesta tabela e linkar em [prompts.md](../prompts.md)

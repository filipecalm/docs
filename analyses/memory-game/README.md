1. Produto **em produção inicial** — temas grátis + custom R$ 19,90 + AdMob; infra e Play Store pipeline prontos.
2. Concorrente BR mais próximo: **Nossas Lembranças** (R$ 9,90–19,90 web); diferencial é app nativo + temas grátis.
3. Modelo: **freemium híbrido** (ads + pay-per-game); assinatura só com dados de recompra.
4. **P0:** funil pós-compra (web auth, Meus jogos, link/QR), compliance, copy de pagamento real.
5. Evitar escalar iOS/B2B antes de conversão ≥1% em escala Android.

# YouTube — Pacote completo (equipe virtual)

**Uso:** gerar título, conceito de thumbnail, roteiro, descrição SEO e tags em uma única execução.  
**Quando usar:** vídeo único com briefing claro; canal com identidade já definida.  
**Resultado esperado:** pacote pronto para gravar/editar, com hook forte e CTA no final.

## Contexto (preencha antes de colar)

- Tópico: `[TOPICO]`
- Formato/referência de tom: `[FORMATO_REFERENCIA]` (ex.: documentário 12 min, MrBeast, canal educativo BR)
- Idioma da saída: `[IDIOMA]` (default: português BR)
- Público-alvo: `[PUBLICO_ALVO]` (opcional)

## Texto do prompt

```
You are my full YouTube team: strategist, creative director, scriptwriter, SEO expert, and thumbnail designer.

Task: Build a complete YouTube content package on: [TOPICO]

Deliver:
1. 5 viral, curiosity-driven titles (in [IDIOMA])
2. MrBeast-style thumbnail concept (strong visual tension) — describe layout, expressions, text overlay, colors
3. Full script (hook -> conflict -> resolution -> CTA) — optimize for retention every 30 seconds
4. Optimized description (SEO keywords, value proposition, timestamps, CTAs)
5. Hashtags + keyword tags (mix broad + niche)

Tone/Format: Match pacing, tone, and style to: [FORMATO_REFERENCIA]
Audience: [PUBLICO_ALVO]

Guidance:
- Think like a YouTube growth strategist
- Optimize for retention, engagement, and shareability
- Hook must be irresistible in the first 5 seconds
- Structure tight; every section earns the next
- End with a clear CTA (subscribe, comment prompt, or next video)
```

## Variações

**Só títulos + thumbnail:**

```
Using the YouTube team prompt above, deliver only items 1 and 2 for: [TOPICO]
```

**Roteiro a partir de título escolhido:**

```
Using the YouTube team prompt above, deliver only item 3.
Title locked to: [TITULO_DO_VIDEO]
Length target: [DURACAO_MINUTOS] minutes spoken.
```

**Tradução para PT-BR:**

```
Execute the full package above but write all deliverables in Brazilian Portuguese.
```

## Checklist

- [ ] `[TOPICO]` específico o suficiente (não "finanças", mas "como o PIX mudou os bancos em 2024")
- [ ] `[FORMATO_REFERENCIA]` definido
- [ ] Se canal existente, mencionar nicho e tom no início da conversa

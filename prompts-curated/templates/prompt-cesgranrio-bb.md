# CESGRANRIO — Banco do Brasil (questões)

**Uso:** gerar questões de múltipla escolha no padrão CESGRANRIO para BB Escriturário / Agente Comercial.  
**Quando usar:** estudo por tema (conhecimentos bancários, vendas, informática, etc.).  
**Resultado esperado:** 5 questões + gabarito comentado, distrator plausível, pegadinhas de produto.

**Fonte:** @profandresocrates PROMPT 4 — `Screenshot_20260121-225759.png`

## Contexto

- Tema: `[TEMA]` (ex.: CDB vs poupança, cartão de crédito, ética no atendimento)
- Cargo: `[CARGO]` (Escriturário | Agente Comercial — default: Escriturário)
- Quantidade: `[QUANTIDADE]` (default: 5)

## Texto do prompt

```
CONTEXTO:
Você é especialista em elaboração de questões de concurso público para a banca CESGRANRIO, especificamente para o concurso do Banco do Brasil — cargos Escriturário e Agente Comercial.

TAREFA:
Crie [QUANTIDADE] questões de múltipla escolha (5 alternativas cada) no formato CESGRANRIO sobre o tema: [TEMA]

ESPECIFICAÇÕES OBRIGATÓRIAS:
- Conteúdo: priorizar Conhecimentos Bancários, Atendimento, Vendas e Ética
- Incluir quando couber: produtos bancários, crédito, investimentos, cartões
- Conceitos: CDB, poupança, capitalização, seguros, consórcios
- Enunciados práticos e situacionais (2–4 linhas)
- Alternativas baseadas no dia a dia bancário
- Distratores: erros sutis em taxas, prazos ou características de produtos
- Pegadinhas: diferenças entre produtos similares, rentabilidades, garantias FGC
- Gabarito: distribuição equilibrada entre a, b, c, d, e
- Estrutura: numeradas 1 a N, 5 alternativas cada
- Ao final: gabarito comentado com explicação prática de cada alternativa errada

Nível: básico-intermediário ([CARGO] BB).
```

## Variações

**Simulado misto (vários temas):**

```
Gere 10 questões CESGRANRIO BB: 3 Conhecimentos Bancários, 3 Vendas, 2 Informática, 2 Atualidades financeiras.
```

**Modo prova (sem gabarito até pedir):**

```
Entregue só as questões numeradas. Aguarde eu responder antes de mostrar o gabarito comentado.
```

# YouTube — Ideias virais (brainstorm)

**Uso:** gerar 20 ideias de vídeo com potencial de viralização e retenção.  
**Quando usar:** início de calendário editorial, teste de nicho, ou quando estiver sem pauta.  
**Resultado esperado:** lista ranqueada com título provisório, gancho e justificativa algorítmica.

## Contexto

- Nicho: `[NICHO]`
- Público-alvo: `[PUBLICO_ALVO]`
- Idioma: `[IDIOMA]` (default: português BR)

## Texto do prompt

```
Atue como um estrategista de crescimento no YouTube com mais de 15 anos de experiência ajudando criadores a escalar canais no nicho/segmento [NICHO].

Sua missão é criar 20 ideias de vídeos com alto potencial de viralização, combinando gatilhos emocionais, curiosidade e otimização para o algoritmo.

Público-alvo: [PUBLICO_ALVO]
Idioma das ideias: [IDIOMA]

Para cada ideia, entregue:
1. Título provisório (até 12 palavras)
2. Gancho dos primeiros 5 segundos
3. Por que o algoritmo pode impulsionar (retenção, CTR, sessão)
4. Formato sugerido (listicle, documentário, reação, etc.)
5. Dificuldade de produção (baixa/média/alta)

Ao final, ranqueie as top 5 com justificativa.
```

## Variações

**Foco em shorts:**

```
Mesmo prompt acima, mas todas as ideias devem funcionar como YouTube Shorts (< 60s) com gancho imediato.
```

**Canal sem rosto:**

```
Mesmo prompt acima. Restrição: nenhuma ideia exige rosto ou presença do criador na câmera.
```

**Só curiosidade negativa:**

```
Mesmo prompt acima. Priorize erros, fracassos, segredos, punições e conspirações — sem inventar fatos.
```

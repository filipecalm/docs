# YouTube — Roteiro magnético (irônico / sarcástico)

**Uso:** roteiro longo com tom envolvente, piadas e comentários sarcásticos — estilo documental opinativo.  
**Quando usar:** após definir título e referência de canal.  
**Resultado esperado:** roteiro falado, com hook nos primeiros segundos e retenção a cada bloco.

> Baseado em `Screenshot_20260225-133309.png` (screenshot truncado). O template completa estrutura e entregáveis ausentes no original.

## Contexto

- Canal de referência (tom narrativo): `[CANAL_REFERENCIA]`
- Tema central: `[ASSUNTO]`
- Título do vídeo: `[TITULO_DO_VIDEO]`
- Quantidade de palavras: `[QUANTIDADE_PALAVRAS]` (default: 1000)
- Idioma: `[IDIOMA]` (default: português BR)

## Texto do prompt

```
Quero que você assuma o papel de roteirista profissional de YouTube, criando roteiros envolventes, dinâmicos e bem estruturados, no mesmo estilo narrativo do canal [CANAL_REFERENCIA].

Diretrizes:
- Roteiro com [QUANTIDADE_PALAVRAS] palavras, em [IDIOMA]
- Tema central: [ASSUNTO]
- Título do vídeo: [TITULO_DO_VIDEO]

Estrutura narrativa:
1. HOOK (0:00–0:30): introdução chamativa e irônica que quebra expectativa e cria curiosidade imediata
2. CONTEXTO: apresente o cenário com tom sarcástico mas informativo
3. DESENVOLVIMENTO: fatos em ordem lógica, com piadas, comparações inusitadas e comentários sarcásticos — sem perder precisão
4. VIRADAS: pelo menos 2 reengajamentos ("mas aqui é que fica interessante...", "e ninguém fala disso...")
5. CONCLUSÃO: síntese memorável + pergunta para comentários
6. CTA: inscrição ou próximo vídeo sugerido

Regras de escrita:
- Frases curtas para narração; marque [PAUSA] onde necessário
- Evite jargão sem explicar com analogia
- Não invente dados — se faltar fonte, sinalize [VERIFICAR]
- Tom: inteligente, irônico, nunca preachy

Entregue o roteiro em blocos com timestamp estimado (mm:ss) e uma linha de "por que isso retém" a cada bloco.
```

## Variações

**Adaptar para narração IA:**

```
Após gerar o roteiro acima, adapte para TTS: frases mais simples, menos ironia ambígua, ritmo uniforme.
```

**Vídeo sem rosto + storyboard:**

```
Após o roteiro, quebre cada bloco em 1–3 sugestões visuais (stock/Canva) com timing.
```

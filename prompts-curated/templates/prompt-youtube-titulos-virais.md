# YouTube — Títulos virais (clickbait controlado)

**Uso:** gerar títulos de alto CTR no estilo de canais de referência, sem sair do nicho.  
**Quando usar:** após escolher o tema do vídeo; antes de gravar ou gerar roteiro.  
**Resultado esperado:** 10–15 títulos curtos, com palavra-chave em caixa alta e curiosidade negativa quando fizer sentido.

## Contexto

- Nicho: `[NICHO]`
- Canal de referência (tom/estilo): `[CANAL_REFERENCIA]`
- Idioma: `[IDIOMA]` (default: português BR)
- Assunto do vídeo (opcional): `[ASSUNTO]`

## Texto do prompt

```
Quero que você atue como um criador de títulos virais para YouTube no nicho de [NICHO], no estilo do canal [CANAL_REFERENCIA].

Assunto do vídeo (se aplicável): [ASSUNTO]

Siga estas orientações:
- Os títulos devem estar em [IDIOMA]
- Gere 15 opções
- Use pelo menos uma palavra em CAIXA ALTA por título (ex: SEGREDO, NUNCA, CRUEL)
- Inclua expressões superlativas quando couber: "AS MAIS INCRÍVEIS...", "VOCÊ NUNCA imaginaria..."
- Explore curiosidade negativa: erros, punições, fracassos, segredos, conspirações — sem inventar fatos
- Máximo 12 palavras por título
- Evite títulos enganosos que o conteúdo não sustente

Ao final, indique os 3 melhores com previsão de CTR (alta/média) e risco de clickbait tóxico (baixo/médio/alto).
```

## Variações

**A/B test (2 estilos):**

```
Gere 8 títulos no estilo [CANAL_REFERENCIA] e 8 no estilo educativo/sóbrio para o mesmo [ASSUNTO].
```

**SEO + click:**

```
Além das regras acima, inclua a palavra-chave principal "[PALAVRA_CHAVE]" naturalmente em pelo menos 5 títulos.
```

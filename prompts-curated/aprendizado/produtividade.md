# Aprendizado e produtividade — 6 prompts

**Uso:** estudar tópicos em profundidade, montar planos de skill, resolver bloqueios mentais e construir modelos mentais.  
**Quando usar:** aprendizado autodidata, preparação para entrevistas, ou destravar hábitos.  
**Resultado esperado:** explicação estruturada, plano de ação ou framework reutilizável.

**Fonte:** `Screenshot_20260116-170111.png`, `170116.png`, `170119.png`

Substitua `[TOPICO]`, `[HABILIDADE]` ou `[PROBLEMA]` antes de colar.

---

## 1. Entender como um gênio

```
Quero entender [TOPICO] como se eu fosse um gênio. Divida o conceito usando analogias avançadas, aplicações do mundo real, contraexemplos e múltiplas perspectivas. Depois teste meu entendimento com 5 perguntas de nível especialista (aguarde minhas respostas antes de corrigir).
```

---

## 2. Dominar habilidade com recursos grátis

```
Suponha que você é um mestre de [HABILIDADE] com mais de 20 anos de experiência. Reprograme o processo que te levou até lá e monte um plano dia a dia para alcançar nível intermediário-avançado o mais rápido possível usando apenas recursos gratuitos ou de baixo custo.

Inclua: marcos semanais, recursos específicos (links quando possível), armadilhas comuns e como medir progresso.
```

---

## 3. Resolver bloqueios mentais

```
Tenho lutado com [PROBLEMA] (ex.: procrastinação, burnout, medo de falhar). Analise como um cientista cognitivo: causas prováveis, padrões comportamentais, gatilhos de ambiente. Desenvolva um loop de hábito concreto (gatilho → ação mínima → recompensa) para reduzir o problema em 30 dias.
```

---

## 4. Confusão → clareza

```
Não entendo [TOPICO]. Explique passo a passo usando metáforas, descrições visuais e exemplos do mundo real. Depois crie um atalho mental ou modelo de 1 frase que eu possa usar para lembrar para sempre.
```

---

## 5. Nível doutorado

```
Me ensine [TOPICO] como se eu estivesse me preparando para um doutorado. Comece pelos princípios fundamentais, explique as teorias centrais com evolução histórica, e liste os 5 artigos/livros essenciais para aprofundar (com 1 linha sobre por que cada um importa).
```

---

## 6. Modelo mental / decisão

```
Estou tentando dominar [TOPICO]. Construa um modelo mental ou framework de decisão personalizado que simplifique como abordar, avaliar progresso e melhorar ao longo do tempo — como faria um profissional. Formato: checklist + 3 perguntas que devo fazer antes de cada sessão de estudo/prática.
```

## Variações

**Só teste (após prompt 1 ou 5):**

```
Aplique 10 perguntas progressivas sobre [TOPICO]. Aumente a dificuldade. Corrija com referências.
```

**Plano 30 dias (combina 2 + 6):**

```
Use os prompts 2 e 6 em sequência para [HABILIDADE]. Entregue calendário de 30 dias unificado.
```

# Desenvolvimento e engenharia

Construir software, segurança e análise técnica com agente.

## Desenvolvimento de Aplicativos

5 prompts para criar aplicativos do zero: arquitetura, desenvolvimento, design, monetização e lançamento.

| # | Prompt | Descrição |
|---|--------|-----------|
| 1 | Arquiteto do Aplicativo | Planejamento completo: estrutura, MVP, stack tecnológico, custo de infraestrutura e design system |
| 2 | Desenvolvedor Full Stack | Construção etapa por etapa com código funcional comentado e configuração de deploy gratuito |
| 3 | Designer de Interface | Design system, paleta, tipografia, componentes, tela principal e versão mobile |
| 4 | Monetizador do App | Modelo de monetização com planos, preços e cálculo para R$10k/mês |
| 5 | Lançador do App em 10 Minutos | Publicação, primeiros usuários e validação de tração em 7 dias |
